Navigation - Bojen und Korridore

Zur Unterstützung der Navigation, insbesondere das Befahren definierter Korridore, können sogenannte Bojen angelegt werden. Dies sind Punktobjekte im 2D-Betrachtungsfenster, die Neben der Koordinate noch eine bestimmte Farbe und Beschreibung erhalten können.

Eine Boje kann über zwei Wege erzeugt werden:

  1. Im 2D-Betrachtungsfenster die Tastenkombination Strg+B verwenden. Der Mauszeiger ändert sich und es wird eine Boje mit der linken Maustaste platziert.
  2. Im Sonarschnitt die Tastenkombination Strg+B verwenden. Der Mauszeiger ändert sich und es wird eine Boje mit der linken Maustaste platziert. Voraussetzung ist eine mit der Sonarlinie verknüpfte Koordinate.
  3. Im Projektbaum die "Buoylist" öffnen.

Mit den Schaltflächen "Add" und "Remove" können Bojen hinzugefügt und auch gelöscht werden. Analog werden Verbindungen zwischen Bojen angelegt (s.u.). Mit "Save" und "Load" wird die Bojenliste samt ihren Verbindungen in eine separate XML-Datei gespeichert bzw. aus einer solchen geladen. Das akutelle Projekt bleibt davon unberührt. Die Bojen werden ebenfalls mit dem Projekt mitgespeichert und geladen. Die Schaltfläche "Clear" löscht die Listen.

Durch Auswahl einer Boje in der oberen Liste ändert sich die Detailbetrachtung rechts - es werden die Eigenschaften der Boje angezeigt.

Coordinate Die Koordinate der Boje (siehe Koordinaten-Editor)
Depth Die Tiefe der Boje (bezogen auf Wasseroberfläche)
Color Die Darstellungsfarbe
Description Eine ergänzende Beschreibung
ID Eine eineindeutige Nummer (innerhalb der Bojenliste)
Type Eine frei wählbare Typ-Nummer für die Boje

Zwischen den Bojen können Verbindungslinien angezeigt werden. Diese müssen vom Anwender spezifiziert werden. Dafür gibt es die zweite Liste "Connections". Beim Hinzufügen einer neuen Bojenverbindung wird der Anwender aufgefordert, die Start- und Zielboje auszuwählen. Die Verbindung trägt als Standardeinstellung eine Beschreibung mit den IDs der beiden Bojen in der Form "[Startbojen-ID] - [Zielbojen-ID]". Diese kann nachträglich geändert werden.

 

Durch Auswahl einer Bojenverbindung in der untern Liste ändert sich die Detailbetrachtung rechts - es werden die Eigenschaften der Verbindung angezeigt.

Start Buoy Die Startboje (nicht editierbar)
End Buoy Die Zielboje (nicht editierbar)
Catch Corridor Width Die absolute Fangweite für die Navigation (>= Korridorbreite)
Corridor Width Eine optionale Korridorbreite (absolute Breite)
Transparency Transparenz des Korridors in der 2D-Darstellung
Color Farbe der Verbindung
Description Eine ergänzende Beschreibung
Highlight Color Farbe der Verbindung (Schiff ist auf Korridor eingerastet)
ID Eine eineindeutige Nummer (innerhalb der Verbindungsliste)
Line Width Die Linienbreite der Verbindung

Ergänzend zur Verbindungsline kann ein dazu zentriert verlaufender Korridor definiert werden (Eigenschaft "Corridor Width"). Das Programm versucht dann während der Aufnahme den nächstgelegenen Korridor zu erfassen und hebt diesen hervor (siehe Bild unten). Zusätzlich wird die Entfernung des Bootes von der Idealline akustisch (abschaltbar) und optisch angezeigt.

 

Der Balken rechts unten stellt die Abweichung von der Idealline dar. Er verfärbt sich von grün in gelb bis zum Rand des Korridors und danach weiter in rot bis zur durch die Fangweite eingestellte Entfernung zur Idealline. Die akustische Signalisierung steigert die Tonhöhe mit wachsendem Abstand zur Idealline (keine Unterscheidung der Lage) und verstummt beim Überschreiten der doppelten Entferung zur Verbindung.
