Formulare - Einstellungen

In diesem Fenster können die meisten der für das Programm wichtigen Einstellungen vorgenommen werden. Diese werden in einer in verchiedene Abschnitte unterteilten Liste angezeigt. Dabei steht der Name auf der linken und der Wert auf der rechten Seite.

Über die Schaltflächen am unteren Rand kann die Liste an Einstellungen in einer seperaten XML-Datei gespeichert bzw. aus dieser geladen werden. Generell gilt, dass sämtliche Einstellungen mit dem jeweiligen Projekt gespeichert werden. Somit dienen die o.g. Schaltflächen lediglich dem Verwalten von Standardeinstellungen, welche Sie für neue Projekte übernehmen möchten.
