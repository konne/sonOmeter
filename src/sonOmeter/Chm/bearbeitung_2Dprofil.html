Bearbeitung - 2D Profil erzeugen

Entlang einer Bojenverbindung kann mit dieser Funktion ein neuer Datensatz (Record) mittels Interpolations der darin enthaltenen Sonardaten erzeugt werden. Dieser Datensatz kann sowohl direkt exportiert als auch als neuer Record in das Projekt aufgenommen werden. Im Folgenden werden alle Einstellungen anhand der Abbildung erklärt:

Die Liste im oberen Bereich stellt alle verfügbaren Bojenverbindungen dar. Es muss mindestens eine dieser Verbindungen ausgewählt sein, damit mit eine Interpolation entlang dieser Linie durchgeführt werden kann. Es können auch mehrere Linien ausgewählt werden, welche dann jeweils für sich berechnet werden. Grundsätzlich ist es wichtig, dass im Projektfenster alle Records und / oder Sonare mit einem Haken versehen sind, die für die Interpolation heran gezogen werden sollen. Es sind nun folgende Optionen wählbar:

One File Erzeugt nur eine Export-Datei bei mehreren gewählten Verbindungen.
Normalize Sonar Lines Die im Korridor einer Bojenverbindung gefundenen Sonarlinien werden auf die Linie der Bojenverbindung normalisiert.
Normalize Manual Points Die im Korridor einer Bojenverbindung gefundenen Manuellen Punkte werden auf die Linie der Bojenverbindung normalisiert.
Merge Manual Points Für den Fall einer Abtastung (siehe Sampling Resolution) werden damit auch die Manuellen Punkte zusammengefasst.
Manual Points dominate Sind in einem Abtastsektor sowohl Sonarlinien als auch Manuelle Punkte vorhanden, werden letztere verwendet (siehe Sampling Resolution).
Add Buoys as Manual Point Die Start- und Endbojen der Verbindung werden mit als Manuelle Punkte an die entsprechenden Stellen eingefügt.
Sampling Resolution Dieser Wert gibt die Auflösung der Abtastung in Metern vor. Der Wert 0 wird als Deaktivierung dieser Funktion interpretiert.

Die Schaltflächen am unteren Rand des Fensters erzeugen eine oder mehrere Export-Dateien ("Export Data"), einen oder mehrere neue Record ("New Records") bzw. dienen dem Verlassen des Fensters.
