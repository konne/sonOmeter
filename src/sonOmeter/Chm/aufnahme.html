Datenaufnahme

Die Aufnahme kann entweder über das Hauptmenü unter dem Menüpunkt "Record", oder über das Formular "Controls" gesteuert werden. Der Vorteil der zweiten Variante liegt in der deutlich besseren Bedienbarkeit großer Schaltflächen, besonders unter den schwierigen Bedingungen auf dem Boot.

Start / Stop Starten und Stoppen einer Aufnahme
Save Speichern
Start Track Nur Spurverfolgung aktivieren
Next Neuen Record anlegen
Config Server Konfiguration des sonOmeter Server Programms
Manual Input Manuelle Eingabe von Punkten

Die Schaltfläche "Start" wandelt sich nach Betätigung in "Stop". Somit können Aufnahmen intuitiv gestartet und abgeschlossen werden. Mit der Schaltfläche "Next" wird ein kurz aufeinander folgendes "Stop"-"Start"-Kommando nachempfunden, um die Aufnahme in einem neuen Record fortzusetzen. Mit "Start Track" wird lediglich die Position ausgewertet und in der 2D-Ansicht dargestellt. Es werden dabei keine Daten aufgenommen.
