Classes

Class Description
DockContainer The DockContainer class is derived from the standard framework class System.Windows.Forms.Panel. It provides the basic functions to dock windows including the whole panel management.
DockEventArgs DockEventArgs is used with DockEventHandler to report docking requests of a moving DockWindow to DockManagers.
DockManager This class is derived from the DockDotNET.DockContainer class to extend its features. Use this class as top-level container in the destination windows of your application.
DockPanel This class is derived from the standard framework class System.Windows.Forms.Panel. It is used as final container of the window's controls and is transferred between DockContainer objects and the window.
DockWindow The DockWindow class is derived from the standard framework class System.Windows.Forms.Form. It prepares the window for docking with the help of an own container of the DockPanel type.
