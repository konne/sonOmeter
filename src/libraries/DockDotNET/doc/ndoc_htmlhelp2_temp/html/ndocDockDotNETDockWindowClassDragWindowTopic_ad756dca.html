DockWindow.DragWindow Event

Occurs, when the window is moved over the screen.

Event Data

The event handler receives an argument of type DockEventArgs containing data related to this event. The following DockEventArgs properties provide information specific to this event.

PropertyDescription
DockType Gets the needed target container type.
Point Gets the point where to dock the window.
Release Gets the state of the dock process. True, if dock is finally performed.
Target Gets or sets the target retrieved by one of the containers.
